# ACS Public Content Kit

Status: redacted public-source material
Last updated: 2026-05-07

This directory stores sanitized material for public ACS articles, website pages,
community posts, and contribution calls.

Use these files as reusable source material:

- `acs-public-material-pack.md`: main story, positioning, article angles, and
  matrix distribution snippets.
- `acs-redacted-collaboration-cases.md`: sanitized collaboration cases.
- `acs-anti-pattern-library.md`: public anti-pattern summaries.
- `acs-public-screenshot-plan.md`: screenshot list, captions, and redaction
  notes.
- `screenshots/`: generated public-safe screenshot-style images based on
  sanitized ACS excerpts.

Redaction source of truth:

- `docs/case-study-redaction.md`

Do not add customer names, private repository URLs, local paths, secrets,
unreleased commercial details, or raw private screenshots to this directory.
