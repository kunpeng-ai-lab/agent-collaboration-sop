# OpenClaw Adapter

Use OpenClaw agents with explicit role prompts:

```text
You are participating in Agent Collaboration SOP.
Role: <Executor/Reviewer>
Project Short Name:
Owner Title:
Do not proceed beyond the approved phase.
```

Recommended:

- keep one session per role
- store handoff files under `docs/`
- store evidence under `evidence/`
- avoid letting an executor edit reviewer reports
