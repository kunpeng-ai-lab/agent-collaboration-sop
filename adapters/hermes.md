# Hermes Adapter

Use Hermes for messaging or workflow-connected agents by keeping the role state explicit in every task payload:

```text
Protocol: Agent Collaboration SOP
Role: <Executor/Reviewer>
Project:
Project Short Name:
Current Phase:
Allowed Scope:
Non-Scope:
Owner Approval Required For:
```

Reviewer tasks should include the reviewer quality bar and require a structured reviewer report.
